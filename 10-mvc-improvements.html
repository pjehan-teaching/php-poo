Améliorations

Amélioration de la couche d'accès aux données

  1. Définir un nom de table par défaut à partir du nom de l'entité dans la classe Entity
  2. Ajouter une méthode getOne() dans la classe Entity
  3. Modifier le code existant pour permettre d'exécuter des requêtes de type INSERT, UPDATE ou DELETE

Amélioration des controllers

  1. Créer une classe Request permettant de gérer les données de $_POST et $_GET
  2. Créer une classe Response permettant de gérer les réponses HTTP (200, 404...)
  3. Créer une classe Session permettant de gérer les données de $_SESSION
  4. Créer une classe FlashMessage permettant de gérer les messages flash
