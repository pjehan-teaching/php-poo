Construire une architecture MVC

Améliorations

Amélioration du routeur

  1. Améliorer la gestion des méthodes HTTP (GET, POST, PUT, DELETE) en utilisant des énumérations
  2. Ajouter une méthode pour générer des URLs à partir du nom de la route et des paramètres

Amélioration de la couche d'accès aux données

  1. Définir un nom de table par défaut à partir du nom de l'entité dans la classe Entity
  2. Ajouter une méthode getOne() dans la classe Entity
  3. Améliorer la méthode getAll() afin de pouvoir filtrer les données, les trier et limiter le nombre de résultats
  4. Modifier le code existant pour permettre d'exécuter des requêtes de type INSERT, UPDATE ou DELETE

Amélioration des controllers

  1. Personnaliser le message d'erreur lorsque la vue appelée par la méthode render n'existe pas
  2. Créer une classe Request permettant de gérer les données de $_POST et $_GET et injecter cette classe lors de l'appel d'une méthode d'un controller.
  3. Créer une classe Response permettant de gérer les réponses HTTP (200, 404...). Les controllers devront donc retourner un objet Response et le front controller se chargera de retourner le contenu de l'objet.
  4. Créer une classe Session permettant de gérer les données de $_SESSION
  5. Créer une classe FlashMessage permettant de gérer les messages flash
Les classes Request et Response pourraient être basées sur les interfaces du PSR-7
